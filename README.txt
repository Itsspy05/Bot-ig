# IG Views Bot - README

## Introduction
This script is designed to automatically boost Instagram Reels views. It uses automated actions to interact with Instagram's platform, simulating user activity to increase views for your reels. This project is intended strictly for educational purposes only.

## Features
- Automated Instagram Reels views boosting.
- Simple setup with GitHub Codespace or Railway.
- No manual interaction required after setup.
- Customizable views count and targeting.
- 24/7 running capability when deployed on Railway.

## Requirements
- Python 3.x+
- Pip (Package Installer for Python)
- Instagram Account (with proper credentials)
- GitHub account (for hosting and deployment)
